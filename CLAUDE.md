# Notions — инструкции для Claude

Общение на русском. Документация проекта: `AGENT.md`.

## Суб-агенты

Проект использует специализированных агентов в `.claude/agents/`. Делегируй задачи агентам вместо выполнения напрямую:

| Агент | Модель | Когда использовать |
|-------|--------|--------------------|
| `prompt-architect` | opus | Сложная задача → сначала декомпозиция и план |
| `frontend-dev` | sonnet | Любые изменения в `frontend/src/` (компоненты, хуки, стили) |
| `backend-dev` | sonnet | Любые изменения в `backend/` (роутеры, сервисы, storage, миграции) |
| `deployer` | haiku | git commit/push, проверка деплоя, диагностика Render |

## Порядок работы

1. **Простая задача** (затрагивает один домен) → делегируй напрямую `frontend-dev` или `backend-dev`
2. **Сложная задача** (несколько доменов / неясный план) → сначала `prompt-architect` для плана, затем агенты по плану
3. **Независимые подзадачи** → запускай агентов параллельно
4. **Деплой** → `deployer` после проверки линтерами

## Линтеры (проверять перед коммитом)

```
uv run ruff check backend/
cd frontend && npx eslint src/ && npx tsc --noEmit
```
